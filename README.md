## ğŸš€ Quick Start

### Prerequisites

- **Node.js** v18+ ([Download](https://nodejs.org/))
- **MySQL** v8.0+ ([Download](https://dev.mysql.com/downloads/mysql/))
- **Google Cloud** account (for Gemini AI API key)
- **Firebase** project (for Google OAuth)

### 1ï¸âƒ£ Database Setup

```sql
-- Run in MySQL Workbench or command line
CREATE DATABASE smart_fitness;
```

### 2ï¸âƒ£ Backend Setup

```bash
cd Backend
npm install
```

Create `.env` file in Backend folder:

```env
# Server
PORT=3000

# Database
DB_HOST=localhost
DB_USER=root
DB_PASSWORD=your_mysql_password
DB_NAME=smart_fitness

# Security
JWT_SECRET=your_super_secret_jwt_key_here

# AI (Get from Google AI Studio)
GEMINI_API_KEY=your_gemini_api_key

# Email (Gmail App Password)
EMAIL_USER=your_email@gmail.com
EMAIL_PASS=your_gmail_app_password
```

Start the server:

```bash
npm run dev     # Development with auto-reload
# OR
npm start       # Production
```

Backend runs at: `http://localhost:3000`

### 3ï¸âƒ£ Frontend Setup

```bash
cd Frontend
npm install
npm start
```

Frontend runs at: `http://localhost:4200`

### 4ï¸âƒ£ Firebase Setup (for Google OAuth)

1. Create a project at [Firebase Console](https://console.firebase.google.com/)
2. Enable Google Sign-In under Authentication
3. Create `Frontend/src/app/config/firebase.ts`:

```typescript
export const firebaseConfig = {
  apiKey: "your-api-key",
  authDomain: "your-project.firebaseapp.com",
  projectId: "your-project-id",
  // ... rest of config
};
```

4. Download service account key and save to `Backend/src/config/serviceAccountKey.json`

## ğŸ”§ Troubleshooting

### Backend Issues

| Problem | Solution |
|---------|----------|
| Database connection failed | Check MySQL is running & `.env` credentials |
| Port 3000 in use | Change PORT in `.env` or kill existing process |
| Gemini API errors | Verify GEMINI_API_KEY is valid |

### Frontend Issues

| Problem | Solution |
|---------|----------|
| Port 4200 in use | `ng serve --port 4300` |
| API calls failing | Ensure backend is running on port 3000 |
| Google login not working | Check Firebase config |

### Email Issues

| Problem | Solution |
|---------|----------|
| OTP not sending | Enable "Less secure apps" or use App Password |
| Gmail blocked | Use Gmail App Password (2FA required) |
